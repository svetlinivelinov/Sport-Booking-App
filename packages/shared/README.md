# packages/shared

Shared cross-platform package for web and mobile.

## Contents

- Shared TypeScript types and DTOs
- Validation schemas
- Constants and enums (game format, roles, status values)
- API contract helpers

## Folder Structure

- `src/domain`: sport-agnostic entities, enums, and rule contracts
- `src/theme`: shared tokens for colors, typography, spacing
- `src/constants`: auth and paging constants
- `src/types`: API-level shared response types
- `src/utils`: small pure helpers safe for web and mobile
- `src/validation`: lightweight guards and validators
